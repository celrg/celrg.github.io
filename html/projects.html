Below is a description of some of the projects that I have been a part of. See them on GitHub.

// WEB CRAWLER

This project entailed traversing through a simulated network of facebook profiles in order to find 5 "secret keys". The Web Crawler sent and processed the results of HTTPS requests without using existing libraries as per the contraints of the project. This entailed constructing requests and parsing the response HTML for cookies, links and flags. This implementation utilizes Keep-Alive connection for performance benefits.

SKILLS AND TECHNOLOGIES: Graph Traversal , HTTPS requests, HTML, authentication analysis, socket programming, Python, PyCharm

// DISTRIBUTED KEY-VALUE DATABASE

Implemented the raft consesus algorithm in order to create a distributed key-value database. Project 'store' implements the algorithm as described on Stanford University's paper detailing the structure of the system and the necessary processes to maintain it.

SKILLS AND TECHNOLOGIES: concurrent processes, socket programming, raft, python, PyCharm

// TWITTER LYRIC BOT

Interfaces with Genius and Twitter developer tools. Hosted with AWS tools. Grabs lyrics from Genius' databasae using the lyricsgenius database that Interfaces with Genius' public API. The bot is currently under modification to remove the lyricsgenius dependecy and instead directly interface with the API.

SKILLS AND TECHNOLOGIES: Twitter developer tools, Genius developer tools, AWS, HTML, lyricsgenius, socket programming, Python, PyCharm

// IMAGE PROCESSOR

Implemented SOLID principles and MVC design in order to create a fully functioning image-processor that is open to extension but closed to modification. Utilized the builder pattern and function objects for feature flexibility. Worked collaboratively through github in a linux-enviroment.

SKILLS AND TECHNOLOGIES: Object-Oriented Design, MVC, Builder Pattern, Extensible Design, GUI display, GitHub, Processing FileTypes, Java, Intellij

// MARBLE SOLITAIRE

Focused on abstraction in order to create multiple versions of the game Marble Solitaire with as little modification as possible. Implemented MVC design.

SKILLS AND TECHNOLOGIES: Object-Oriented Design, MVC, Java, Intellij
